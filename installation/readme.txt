Install this tool: http://blog.ksoftware.net/tag/ksign/

ksign might be easier to use to create pfx file, etc.

Creating the pfx file:
	1) create a c# project in visual studio
	2) open project properties
	3) select Signing tab
	4) check Sign the assembly
	5) enter the key file name, e.g. easywebrtcieplugin.pfx
		set the password as: priologic
